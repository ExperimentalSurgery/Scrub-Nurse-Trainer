# ApplicationQuitStep

A training step that quits the application when executed.

<p align="center">
    <img alt="ApplicationQuitStep.png" src="../../images/steps/ApplicationQuitStep.png"/>
</p>


## Usage

Use this step when you want to quit the game completely after the step sequence is finished.

## Parameters

### Server Delay

The amount of time to wait on the server before quitting the application.
