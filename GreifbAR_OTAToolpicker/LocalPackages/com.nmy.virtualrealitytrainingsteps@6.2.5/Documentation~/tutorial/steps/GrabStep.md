# GrabStep

A training step that waits for a grabbable object to be held.

<p align="center">
    <img alt="GrabStep.png" src="../../images/steps/GrabStep.png"/>
</p>

## Usage

This step requires ``Auto Hands`` to be installed in the project.

Use this step if you want the user so grab a specific object.

Works in conjunction with [ReleaseGrabStep](GrabStep.md).

## Parameters

### Grabbable

A reference to the Grabbable component that should be grabbed.
