# VirtualAssistantSpeakStep

A training step that allows a virtual assistant to speak

<p align="center">
    <img alt="VirtualAssistantSpeakStep.png" src="../../../images/steps/VirtualAssistantSpeakStep.png"/>
</p>

## Usage

Use this step if you want the virtual assistant to speak to the user. This step handles possible expression animations
that are saved within [``Localized Tts Audio Clip``](#localized-tts-audio-clip) as timestamp + expression string.

## Parameters

### Localized Tts Audio Clip

The localized text-to-speech audio clip that the virtual assistant should use to speak in this training step.
