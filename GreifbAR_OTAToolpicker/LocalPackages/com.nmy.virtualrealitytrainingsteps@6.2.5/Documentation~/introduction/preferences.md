# Preferences

There are some convenient preferences for this framework. To reach the preferences, go to ``Edit/Preferences...`` 
and select ``Virtual Reality Training Steps``.

![preferences.png](../images/preferences.png)

## Step Unity Events

When enabled, each time you add a step component to a GameObject,
a [``BaseTrainingStepUnityEvents`` component](../tutorial/BaseTrainingStepUnityEvents.md) 
is automatically added to the same GameObject.

Default value: ``false``

## Step Debug

When enabled, each time you add a step component to a GameObject,
a [``BaseTrainingStepDebug`` component](../tutorial/BaseTrainingStepDebug.md) 
is automatically added to the same GameObject.

Default value: ``true``
